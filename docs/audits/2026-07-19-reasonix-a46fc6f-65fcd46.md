Windows 本地已覆盖策略与 no-op 测试；真正 Deepin/Arch/WebKit2GTK 2.50 安装态仍需 Linux candidate/真机证据，

### `0a54d504` 会话图片/PDF 导出可靠性

结论：`adopted`。

旧实现用 blob URL 加载含 `foreignObject` 的 SVG，在旧 Chromium/WebView 中可能使 canvas 被污染；长会话还会
超过单画布限制。本批采用：

- 自包含 SVG data URL，CSS 外部 URL 中和，Markdown 图片只允许严格的 raster base64 data URL；
- 外部图片、视频、音频、iframe/object/embed 替换为占位符；
- 按自然内容边界分页，PNG 逐片编码，PDF 每页独立 raster，避免超大画布和空白尾页；
- 多段 PNG 由 Go 后端一次性 staging：编号 sibling 不覆盖已有文件，完整写入/同步后独占发布；
- commit 失败只回滚本次创建且 inode 身份仍匹配的文件，避免删除并发替换目标；
- Native dialog 选择路径不做静默重写，错误只返回安全文件名，成功/失败均显示用户反馈。
